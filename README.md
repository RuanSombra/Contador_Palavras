SINGLETON - O QUE É E POR QUE USAMOS?

É um padrão de projeto que garante que só exista uma única instância da classe no programa inteiro.

Neste projeto foi utilizado esse modelo, por ser:
- Simples;
- Fácil para treinar programação;
- Perfeito para entender como as classes funcionam.

*SINGLETON*

É um padrão de projeto criacional que garante que uma classe tenha apenas uma instância durante a execução de um programa, fornecendo um ponto de acesso global para essa instância.

*POR QUE FOI ESCOLHIDO*

- Simples;
- Bem útil para aprender a entender a programação;
- Implementável em diversos projetos para melhorar a manutenção.
